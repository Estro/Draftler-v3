Draftler-v3
===========

# This version of Draftler is using:

- Grunt
- Compass
- SASS
- mySQL
- Node
- Express.js
- Passport.js
- Bootstrap
- Hogan
- Mustache
- Email js
- Underscore.js
- Bookshelf.js


## Set up

- Ensure you have Node, NPM, mySQL, Compass and Rails installed
- Create config.js in the root and populate with account details
- Run npm install
- Run grunt
- Run node app.js
